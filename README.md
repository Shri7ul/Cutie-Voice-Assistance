## How to run? 

1. Create a virtual environment:

   ```bash
   conda create -n cutie python=3.11 -y
   ```

2. Activate virtual environment:

   ```bash
   conda activate cutie
   ```

3. Install required packages:

   ```bash
   pip install -r requirements.txt
   ```

4. Run the Cutie script:

   ```bash
   python cutie.py
   ```
